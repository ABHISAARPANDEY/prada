# Founder Images

Please add the following images to this directory:

1. `mario-prada.jpg` - Image of Mario Prada (Founder, 1913-1958)
2. `miuccia-prada.jpg` - Image of Miuccia Prada (Creative Director & Co-CEO, 1978-Present)
3. `patrizio-bertelli.jpg` - Image of Patrizio Bertelli (CEO & Co-CEO, 1977-Present)

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 400x500px or larger
- Aspect ratio: Portrait (vertical)
- Quality: High resolution for best display

If images are not available, the component will automatically show a placeholder with the founder's name.
